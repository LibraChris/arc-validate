arc-validate Documentation

The arc-validate repository is the home of all the tools and libraries to create and run validation of ARCs:

  • ARCExpect - easy to use and understand APIs to create and execute validation cases.
  • ARCValidationPackages - package for installing, updating, and executing ARC validation packages.
  • arc-validate/ - CLI tool that offers commands for validating ARCs and managing validation packages.
